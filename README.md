# Installation
Manually
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)" "" --unattended
curl -o ~/.oh-my-zsh/custom/themes/robbyrussell-custom.zsh-theme https://raw.githubusercontent.com/aaron-ak/oh-my-zsh-custom-theme/refs/heads/main/robbyrussell.zsh-theme
```


Use install_custom_theme.sh-script:
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/aaron-ak/oh-my-zsh-custom-theme/refs/heads/main/install_custom_theme.sh)" ""
```
